# [LangChain](https://js.langchain.com/docs/): GPT 框架。 <Badge github="https://github.com/hwchase17/langchainjs" />

> langchain 把 gpt 的能力发挥到了极致。

### 特性

1. Schema: 接口和基础类
2. Models: 多种模型集成
3. Prompts: 提示模板和功能
4. Indexes: 数据处理和准备
5. Memory: 状态保持
6. Chains: 调用序列
7. Agents: 决策代理

### 应用场景

1. [DataBerry](https://www.databerry.ai/): 您可以在几秒钟内创建根据您的数据训练的自定义 AI 聊天机器人。 <Badge github="https://github.com/gmpetrov/databerry" />
2. [Flwise](flowiseai.com): 拖放界面，使用 LangchainJS 构建自定义的 LLM 流程。<Badge github="https://github.com/FlowiseAI/Flowise" />
3. [mendable](https://www.mendable.ai/): 用聊天的方式查看文档。
   - langchain 的官方文档用的就是这个。
4. [helicone](https://github.com/Helicone/helicone): 还不知道是啥。

### 最佳实践

1. 预设 SystemMessage, 可以避免被问出基于 GPT 的信息。
2. [redis memory](https://js.langchain.com/docs/modules/memory/examples/redis)可以持久化对话。
