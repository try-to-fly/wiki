# [Puppeteer](https://pptr.dev/)

## Flag

1. 禁用浏览器右侧【要恢复页面吗】弹窗: `--hide-crash-restore-bubble`
2. 禁用自动化测试提示条: `ignoreDefaultArgs: ["--enable-automation"]`

1. 模拟浏览器特征[via](https://gist.github.com/try-to-fly/7217cecaff28c245552f5808bba7e329)
