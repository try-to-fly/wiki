## 迁移到 EMS

[参考：Pure ESM package](https://gist.github.com/sindresorhus/a39789f98801d908bbc7ff3ecc99d99c)

### 步骤

1. 将 `package.json` 中的 `type` 改为 `module`
2. `tsconfig.json`配置参考

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "Node16",
    "moduleResolution": "Node16",
    "esModuleInterop": true,
    "outDir": "./dist",
    "removeComments": true,
    "forceConsistentCasingInFileNames": true,
    "strict": true,
    "skipLibCheck": true
  }
}
```

3. 修改 import 语句的后缀，都需要以`.js`结尾
