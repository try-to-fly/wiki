# [Vercel](https://vercel.com/)

> Vercel 的前端云为开发人员提供了框架、工作流程和基础设施，以构建更快速、更个性化的网络环境。

### 特色

1. **丰富的免费额度**：提供高达 100GB 的存储和 256MB 的 Postgres 数据库。
2. **多样的模板项目**：一键快速部署，帮助你节省开发时间。
3. **智能自动部署**：兼容 Github、Gitlab、Bitbucket，无缝整合你的代码仓库。
4. **定制化域名**：一旦配置自定义域名，无需任何代理，即可在国内直接访问，并且提供免费的 SSL 证书。
5. **强大的统计分析**：简单的配置就可以开启 Analytics 统计功能，轻松获取访客数据。

### 能力

1. 存储：[Vercel Storage](https://vercel.com/docs/storage)
2. 日志分析： [Vercel Logs](https://vercel.com/docs/logs)

### 缺点

1. 不支持 Docker，无法直接取用其他服务和能力。可以考虑`Render`、`Fly` 方案。
2. `Serverless Function`最长仅支持 10s [参考](https://vercel.com/guides/what-can-i-do-about-vercel-serverless-functions-timing-out)
