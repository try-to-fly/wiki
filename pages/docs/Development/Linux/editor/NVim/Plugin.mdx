# Plugin

## Colorscheme

1. [tokyodark.nvim](https://github.com/tiagovla/tokyodark.nvim): tokyonigth
2. [catppuccin/nvim](https://github.com/catppuccin/nvim): catppuccin-mocha

## LSP

1. [Lspsaga](https://nvimdev.github.io/lspsaga/): 增加 lsp 的提示，类似 vscode 的在浮窗一样显示类型定义。
2. [nvim-lsp-file-operations](https://github.com/antosha417/nvim-lsp-file-operations): 文件重命令自动更新引用路径。

## 语言

### Typescript

1. [typescript-tools](https://github.com/pmizio/typescript-tools.nvim): 加载速度快。
2. [ts-error-translator.nvim](https://github.com/dmmulroy/ts-error-translator.nvim): 翻译 ts 错误，具有可读性。
3. [tsc.nvim](https://github.com/dmmulroy/tsc.nvim): 检查项目的 ts 错误

### Python

1. [venv-selector.nvim](https://github.com/linux-cultist/venv-selector.nvim): 选择虚拟环境。
2. [nvim-dap-python](https://github.com/mfussenegger/nvim-dap-python): Python 调试。

### Prisma

1. [vim-prisma](https://github.com/prisma/vim-prisma)

### Markdown

1. [MDX 高亮](https://github.com/jxnblk/vim-mdx-js): MDX 推荐的插件。注：虽然这个仓库已经归档，但是还是有效果的。

## 模块

1. [nvim-plugin-template](https://github.com/nvimdev/nvim-plugin-template)
