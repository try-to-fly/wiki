### 插件

1. [vim-surround](https://github.com/tpope/vim-surround)
   - `ds`  删除环绕字符，如  `ds"`  删除双引号
   - `cs`  修改环绕字符，如  `cs"'`  把双引号改为单引号
   - `ys`  添加环绕字符，如  `ysiw)`  给单词加上圆括号
   - `S`  在可视模式下添加环绕字符，如选中一段文字后按  `S"`  给它加上双引号
2. [nvim-spectre](https://github.com/nvim-pack/nvim-spectre)
   - `<leader>S`: 打开 Spectre 搜索工具
   - `<leader>sw`: 搜索当前单词或选中文本
   - `<leader>sp`: 在当前文件中搜索当前单词
3. 注释插件: 支持 tsx 代码块。
   - [nvim-ts-context-commentstring](https://github.com/JoosepAlviste/nvim-ts-context-commentstring): 自动推测类型。
   - [tpope/vim-commentary](https://github.com/JoosepAlviste/nvim-ts-context-commentstring/wiki/Integrations#vim-commentary): 代码注释
4. [Telescope](./NVim/Telescope.mdx): 提供快速、交互式的文件浏览、模糊搜索和选择等功能。
