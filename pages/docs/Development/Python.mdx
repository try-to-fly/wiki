### 虚拟环境

1. [venv](https://docs.python.org/3/library/venv.html) 是 Python 3.3 之后内置的虚拟环境工具

2. [conda](https://docs.conda.io/en/latest/) 是 Anaconda 提供的虚拟环境工具

- [miniconda](https://docs.conda.io/projects/miniconda/en/latest/): 推荐安装这个，占用更小。
