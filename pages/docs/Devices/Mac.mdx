### App、Cli

#### 更好的终端命令

1. ls => [exa](https://github.com/ogham/exa)
2. cat => [bat](https://github.com/sharkdp/bat)
3. cd => [zoxide](https://github.com/ajeetdsouza/zoxide): 更智能的 cd
   - 同类：[autojump](https://github.com/wting/autojump)
4. ctrl-r => [mclfy](https://github.com/cantino/mcfly): 更智能的历史命令
5. find => [fd](https://github.com/sharkdp/fd): 简单、快速的 find 替代品

#### 终端增强

1. [oh-my-zsh](https://ohmyz.sh/): 更好用的 zsh
2. [starship](https://starship.rs/): 更好看的命令行提示符
3. [tmux](../Development/Linux/Tmux.mdx): 终端复用

#### zsh 增强

1. [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions): 命令自动补全
2. [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting): 命令语法高亮

#### git 增强

1. [zsh git 插件](https://github.com/ohmyzsh/ohmyzsh/tree/master/plugins/git)：预设了一系列好用的 alias

#### Mac 增强

1. [Raycast](https://raycast.com/): 更好用的 Spotlight，便捷的剪切板管理。同时支持 React 开发插件。
2. [yabai](https://github.com/koekeishiya/yabai): mac 窗口管理
3. [quick look](https://github.com/sindresorhus/quick-look-plugins): 为开发人员提供的快速查看插件列表

#### 其他

1. [yadm](https://github.com/TheLocehiliosan/yadm): 更好用的 dotfiles 管理工具，用于多电脑同步配置文件。

### Homebrew 同步

1. 导出：

```bash
brew bundle dump --describe --force --file="~/Downloads/Brewfile"

```

2. 导入：

```bash
brew bundle --file="~/Downloads/Brewfile"
```
