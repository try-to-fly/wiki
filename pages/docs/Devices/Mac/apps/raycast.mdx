### 插件

#### 内置插件

1. Calculator: 计算器，快速计算常见表达式。
2. Clipboard History: 剪切板历史，最长可以记录 3 个月的内容，非常好用。
3. Window Management: 窗口管理。推荐：Half left、Half right 绑定快捷键。随时随地分屏处理。

#### 第三方

1. [SpeedTest](https://www.raycast.com/tonka3000/speedtest): 网络测速。
2. [GitHub Repository Search](https://www.raycast.com/thomas/github-repository-search): Github 仓库搜索。
