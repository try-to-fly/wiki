1. 原来 flex 的 order 可以修改 dom 的显示顺序

```scss
.button-container {
  display: flex;
  flex-direction: row;

  .npc-entry-btn {
    &:nth-child(1) {
      order: 3;
    }

    &:nth-child(2) {
      order: 1;
    }

    &:nth-child(3) {
      order: 2;
    }
  }
}
```
