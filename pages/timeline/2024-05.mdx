### 05-14

1. 端口=0 会随机分配一个端口

```js
// 写一个express，监听端口0， 打印出端口号
const express = require("express");
const app = express();

app.get("/", (_req, res) => {
  res.send("hello world");
});

const listener = app.listen(0, () => {
  console.log(listener.address().port); // 63610
});
```
