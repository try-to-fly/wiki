#### 推荐软件：

1. [腾讯柠檬清理](https://lemon.qq.com/)

#### 清理步骤

1. 点击开始扫描，清除自动识别的缓存和垃圾信息。
2. 磁盘空间分析：分析系统目录下的资源占用，来定制化删除。

#### 消耗磁盘的大户 App

1. 微信缓存
2. 钉钉缓存
3. yarn 缓存
   1. yarn cache clean
4. pnpm 缓存
   1. pnpm store prune
5. docke 缓存：docker 长期使用，缓存也会非常的大
   1. 清理缓存：docker system prune
   2. 修改磁盘占用：设置 => Resources => Advanced => Disk image size 调整为最小。
